mhttpd is working

mthttpd is working. Excellent! One step closer to total world domination!

mthttpd is an updated version of thttpd v2.27 (Oct 2015) with PHP improvements and most sthttpd 2.27.1 patches (as of 2017). However it will still build with 1989 build tools.

mthttpd is drop in replacement for thttpd or sthttpd, and alternate to PHP SAPI patched thttpd v2.21b

test printenv (confirm confiuration)
test php-cgi presence (edit ./cgi-bin/phpinfo.cgi)
test php.cgi gateway script (edit ./cgi-bin/php.cgi)
test php hello world directly (binfmt_misc or PHP SAPI method)
